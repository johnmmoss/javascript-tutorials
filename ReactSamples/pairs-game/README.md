
## Pairs Game

### Scenarios:

Game Load
* Print out a random order of numbers between 1 to 8 twice for each number
* The buttons will be blank and have no content to start with
* The play again box will be hidden.

First Round
* User clicks first button. The number is revealed and goes to blue color (candidate)
* User clicks second button. The second number is revealed. If numbers match both go to green, else go to red
* After 3 seconds, if red, then numbers go back to being hidden

Second Round
* The above continues go after go, clicking pairs to see if they match
* Correct matchs stay open fails hide after a minute

Extras
* Add in a timer so the game finishes after a minute
* Let user specify number of pairs to have when game starts
